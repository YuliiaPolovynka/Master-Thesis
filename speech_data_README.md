# speech_data.rds

This file contains an R dataset derived from `speech_distr2.csv`.

## Description

- The data is preprocessed and formatted for analysis
- Scenes are ordered correctly within each play

## Source

Created from `speech_distr2.csv` using the R script:
`BEATchart_new.R`

## Usage

This dataset is used in the analysis of beat charts in `BEATchart_new.R` and `BEATchart2_new.R`.
